# Docs

## Installation

To install Sphinx and the required themes, use:

```bash
conda install sphinx sphinx_rtd_theme sphinx-book-theme --channel conda-forge
```

## Style

The docstring style is similar to the [numpydoc style](https://numpydoc.readthedocs.io/en/latest/format.html).

## (Re)Build the Docs

```bash
make clean html
```
